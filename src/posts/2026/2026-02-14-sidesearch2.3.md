- Made it possible to start the assistant using keyboard shortcuts or VoiceOver Magic Tap
- Added support for Assistive Access
- Fixed an issue where speech recognition could not be restarted
- Fixed an issue where closed modals could be re-displayed
- Fixed an issue where multiple windows could be created on the iPad
- Made some other adjustments

Update is available on the [App Store]({{ collections.projects.sidesearch.links.appstore }}) and [AltStore PAL]({{ site.links.altstore }}).
