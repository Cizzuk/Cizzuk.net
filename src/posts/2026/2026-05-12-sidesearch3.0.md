- Added Side Bridge to the assistant
  - You can create and use your own endpoint called Side Bridge, which serves as a bridge between any conversational assistant and Side Search
- Added a feature that allows messages to be automatically entered using URL schemes in responses from the assistant
- Made some other adjustments

Update is available on the [App Store]({{ collections.projects.sidesearch.links.appstore }}) and [AltStore PAL]({{ site.links.altstore }}).
