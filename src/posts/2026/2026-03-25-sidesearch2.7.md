- Added sound effects
- Added a search bar to the chat history
- Enabled resuming the assistant from chat history
- Added automatic keyboard focus when "Start with Mic Muted" is enabled
- Added close and submit buttons to the keyboard
- Removed the "Assistant Screen Size" setting
- Made some other adjustments

Update is available on the [App Store]({{ collections.projects.sidesearch.links.appstore }}) and [AltStore PAL]({{ site.links.altstore }}).
