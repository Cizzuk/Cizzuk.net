- Added a setting to switch assistants
- Added support for conversational assistants
- Added Apple Foundation Models to the assistant
  - Only available on devices that support Apple Intelligence
- Added Google Gemini API to the assistant
  - You need to obtain your own API Key from Google AI Studio
  - You can use the web search feature
- Redesigned the assistant's UI
- Changed the app icon
  - Made it possible to change the app icon
- Changed or removed some settings
- Made some other adjustments

Update is available on the [App Store]({{ collections.projects.sidesearch.links.appstore }}) and [AltStore PAL]({{ site.links.altstore }}).
