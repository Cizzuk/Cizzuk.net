- Restored the default assistant screen size to full screen
- Enabled continuous conversation with assistant in the background
  - To use this, you need to allow notifications
  - If you start a conversation in the background while using a supported assistant, you will be asked to allow notifications
  - URL Based Assistant are not supported
  - If you remain silent for 30 seconds after the assistant replies, the assistant will automatically terminate
- Live Activity now starts when conversing with assistant in the background
- Prevented system Auto-Lock when using speech recognition
- Made some changes to the search engine presets
  - Added Google AI mode
  - Changed Gemini's URL
- Made some other adjustments

Update is available on the [App Store]({{ collections.projects.sidesearch.links.appstore }}) and [AltStore PAL]({{ site.links.altstore }}).
