- Added a feature to record audio notes
- Added a feature to import files
- Removed the "Locked Camera Action" setting and now it is automatically configured according to "Camera Control Action"
- Added a search engine setting for web searches
- Removed the screen width limitation
- Made minor adjustments to the app icon
- Changed the icon for the "Open CBNote" control in the Control Center
- Made some other adjustments
- Enabled some features that were previously unavailable on macOS
  - Enabled camera
  - Added a Translate button to the context menu

Update is available on the [App Store]({{ collections.projects.cbnote.links.appstore }}) and [AltStore PAL]({{ site.links.altstore }}).
