New features:

- Added iCloud support
- Added a feature to search by file name and note content
- Added sorting by name and modification date
- Added a feature to pin notes
- Added an option to immediately open the app in "Locked Camera Action"
- Added an option to customize the file name format

Improvements:

- Disabled Auto-Lock while the camera is in use
- Some editable text files have been set to display in a monospace font
- Fixed an issue where some file formats could not be copied to clipboard
- Implemented countermeasures for issues where the camera takes too long to launch or crashes upon launch
- Fixed other bugs
- Made some other adjustments

Update is available on the [App Store]({{ collections.projects.cbnote.links.appstore }}) and [AltStore PAL]({{ site.links.altstore }}).
