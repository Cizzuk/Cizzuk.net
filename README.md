# Cizzuk.net

私のWebサイトのソースコードです。Eleventyを使用しています。

## Install

```bash
npm install
```

## Build

以下のコマンドで`./_site/`以下に静的Webサイトがビルドされます。

```bash
npm run build
```

ビルド時にはHTML, CSS, JavaScript, XMLのMinifyも行われます。

また、以下のコマンドでローカルサーバーを起動します。

```bash
npm run serve
```
